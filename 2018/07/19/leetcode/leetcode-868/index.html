868. 二进制间距

给定一个正整数 N，找到并返回 N 的二进制表示中两个连续的 1 之间的最长距离。

如果没有两个连续的 1，返回 0 。

思路

N最大为10^9 ,所以2的最大次幂为2^30那么在这30位中我们要计算二进制间距最长的，应该是 一次次遍历找到连续的1记录下连续的1之间的差(i-pre)，若差大于之前计算过的连续1之差len，则将i-pre赋值给len以获得最大的len，遍历下去就可以得到最大的len了~
